¿Qué es FRONTEND?



Es la interfaz de usuario visible y funcionalidad con la que un usuario interactúa directamente, a menudo a través de una pantalla
de computadora, un dispositivo móvil o cualquier otro dispositivo electrónico.


Fundamentos:

HTML (HyperText Markup Language): Es un lenguaje para estructurar y organizar el contenido de una página web mediante etiquetas
que definen elementos como texto, imágenes y enlaces.

CSS (Cascading Style Sheets): Se utiliza para dar estilo y diseño a los elementos HTML, permitiendo controlar colores, fuentes,
márgenes y diseños.

JavaScript: Es un lenguaje de programación que agrega interactividad a las páginas web, permitiendo validar formularios, crear
animaciones y cargar datos de manera dinámica.

Responsive Web Design: Implica diseñar sitios web que se adapten automáticamente a diferentes tamaños de pantalla y
dispositivos, mejorando la experiencia del usuario.

Frameworks y Bibliotecas: Son conjuntos de herramientas predefinidas, como React o Vue.js, que simplifican el desarrollo web
al proporcionar componentes y estructuras reutilizables.

Control de Versiones (Git): Permite rastrear y gestionar cambios en el código fuente, facilitando la colaboración en proyectos
y el seguimiento de revisiones.

Accesibilidad Web: Implica diseñar sitios web de manera que sean utilizables por personas con discapacidades, incorporando atributos
y prácticas de diseño inclusivas.

Herramientas de Desarrollo: Son programas, como las herramientas de desarrollo del navegador, que ayudan a depurar y optimizar el
código web.

Optimización de Rendimiento: Se refiere a las prácticas para mejorar la velocidad y la eficiencia de una aplicación web, como la
carga rápida de páginas y la reducción de tiempos de carga.

Pruebas y Depuración: Involucra verificar y corregir errores en el código mediante pruebas y herramientas de depuración.

Navegadores Web: Es importante conocer cómo los diferentes navegadores interpretan y muestran las páginas web para garantizar la
compatibilidad.

Diseño de Experiencia de Usuario (UX): Se enfoca en crear interfaces de usuario que sean intuitivas y agradables de usar, mejorando
la satisfacción del usuario.
